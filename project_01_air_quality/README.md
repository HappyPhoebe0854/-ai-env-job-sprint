# Project 01: Urban Air Quality Analysis and Prediction

## Project Background
This project focuses on Shanghai air quality data analysis, pollutant factor analysis, data cleaning, visualization, and insight generation.

## Research Questions
- How does Shanghai's air quality change over time?
- Which pollutants contribute most to air quality changes?
- Are there seasonal or station-level differences in air pollution?
- How can air quality data be transformed into useful visual insights?

## Data
Current data source documentation:
- `docs/data_sources.md`
- `docs/field_dictionary.md`

## Planned Workflow
1. Collect and evaluate data sources
2. Build field dictionary
3. Import dataset with Python / Pandas
4. Perform data quality check
5. Conduct exploratory data analysis
6. Build visualizations
7. Summarize findings and recommendations
